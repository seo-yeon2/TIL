## Git의 기본

### Git 이란?

소스코드를 효과적으로 관리하기 위해 개발된 **분산형 버전 관리 시스템**

### Git의 기본

**이력을 관리하는 저장소**

    - 저장소(repository); 파일이나 폴더를 저장해 두는 곳

    -Git이 제공하는 저장소: 원격 저장소(Remote Repository)&로컬 저장소(Local Repsitory)
